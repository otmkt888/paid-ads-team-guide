🚀 接新代理商的完整流程

  • 步驟 1：了解代理商背景
    • 詢問投放經驗（涵蓋哪些市場）
    • 了解轉換成效表現
    • 確認擅長的投放渠道
  • 步驟 2：生成投放連結
    • 使用 ONEHub 生成連結
    • 分為 H5 和 PWA 兩種類型
    🎬 PWA 設置教學影片：

    → 點擊觀看 PWA 完整配置指導影片

  • 步驟 3：埋設追蹤像素
    • 設置 GTM（Google Tag Manager）
    • 記錄並管理 Token
  • 步驟 4：測試轉換事件重要
    • 註冊測試：隨機註冊一筆即可
    • 儲值測試：使用以下專用測試帳號
    測試帳號：
    1️⃣ Username/PSW: paidadstest / paidadstest1
    2️⃣ Username/PSW: paidadstest12 / paidadstest12
    • 儲值事件：Purchase
    • 首次儲值：Start Trial
    • 轉換廣告目標：Start Trial（測試帳號因非首次充值，只能測出 Purchase）
  • 步驟 5：提供素材庫
    • 提供官方素材庫給代理商使用
    • 不建議使用代理商自備素材
    • 若使用自備素材，必須經過審查
  • 步驟 6：廣告上線後的設置
    • 提供每日數據回填表格（使用 廠商成效表 Template）
    • 向代理商索取 Email，申請後台數據權限
    • 填寫權限申請表後，通知 BI 部門的 Jane
    • 提供後台數據表給代理商
    📎 權限申請步驟參考：

    步驟 1：填寫 Email 和 Agency

    步驟 2：完整填寫權限資訊

📈 管理代理商的日常工作

  • 數據監控
    • 確認代理商每日有回填數據
    • 後台數據也必須填寫，否則系統無法抓取成效
  • 成效評估與優化
    • 檢查 CPS、CPR 是否過高或出現異常波動
    • 找機會提升消耗量
    • 砍掉無成效的廣告
  • 溝通與洞察
    • 詢問哪些素材轉換效果好
    • 了解哪些產品賣得好
    • 收集競品投放資訊
  • 技術維護
    • 定期更換 Pixel
  • 財務管理關鍵
    • 定期請款，避免廣告因預算中斷。請款完成要提供水單給代理商對帳
